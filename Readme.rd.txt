# UAE Telecom Customer Churn Analytics & Retention Intelligence

## Project Overview

This project simulates a UAE telecom business environment and analyzes customer churn patterns using Power BI, SQL, Python, and Machine Learning.

The objective is to identify high-risk customers, quantify revenue loss, and provide actionable retention insights.

---

## Tools Used

- Power BI
- DAX
- SQL
- Python
- Pandas
- Scikit-Learn
- Random Forest

---

## Project Workflow

### 1. Synthetic Data Generation

Created a realistic telecom dataset containing:

- 100,000 customers
- Customer demographics
- Service subscriptions
- Contract information
- Revenue metrics
- Churn status

### 2. Data Cleaning

Performed data preparation and transformation using Python.

### 3. SQL Analysis

Conducted business analysis using SQL queries.

Examples:

- Churn by contract type
- Revenue by emirate
- Customer segmentation
- Service adoption analysis

### 4. Machine Learning

Built a Random Forest model to identify key drivers of customer churn.

Top churn drivers include:

- Monthly Charges
- Contract Type
- Tenure
- ARPU
- Payment Method

### 5. Power BI Dashboard

Created a multi-page dashboard containing:

- Executive Summary
- Churn Risk Analysis
- Customer Intelligence
